#### uv 설치과정

- curl -LsSf https://astral.sh/uv/install.sh | sh ⇒ uv 설치
- 맥북의 PATH에 주소 등록해주는 과정 필요 기본 터미널 쉘인 Zsh 설정 파일 (~/.zshrc)에 uv가 있는 폴더 주소 추가
    - (설정 파일에 주소 추가하기) echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
    - (변경된 설정 바로 적용하기) source ~/.zshrc

#### 경로 작업

- ***구체적인 경로를 먼저 선언하고, 포괄적인 동적 경로를 나중에 선언한다.
    - /users/{user_id} 처럼 위에서 일치하는 부분 생기면 user_id = “me”로 처리해버리고 아래에 있는 /users/me 전용 함수까지 내려가서 실행하지 않음
    - (파생) 한마디로, 위에서 일치하면 아래부분 실행 안한다는 거니까, 똑같은 경로 재정의 할 수가 없음.
- Enum 클래스
    1. 아무 값이나 못 들어오게 제한할 수 있음
    2. 오타나 잘못된 요청을 FastAPI가 자동 검증함
    3. Swagger 문서에서 선택 가능한 값이 드롭다운처럼 명확하게 보임
    - class ModelName(str, Enum): → str + Enum = FastAPI가 문자열 선택지로 잘 인식하게 하는 방식
    - model_name == ModelName.alexnet
        - 값 비교
        - 일반적으로 더 익숙한 비교 방식
    - model_name is ModelName.alexnet
        - 같은 Enum 멤버 객체인지 비교
        - Enum에서는 같은 멤버가 하나의 객체처럼 관리되므로 사용 가능
    - model_name.value
        - Enum 멤버의 실제 값 꺼내기
        - 여기서는 "alexnet" 같은 문자열
- @app.get("/files/{file_path}") 이라고 할 때,
    - /files/home/harry/myfile.txt 이런 요청은 ? → 전체 경로를 잡지 못한다.
    - @app.get("/files/**{file_path:path}**") 이렇게 작성하면 file_path라는 파라미터가 /를 포함한 나머지 경로 전체를 받을 수 있다.
